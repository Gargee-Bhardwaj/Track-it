# Habit Tracker App
> A habit tracking app which contains a record of your habits on a daily and weekly basis. You can register and login to maintain your habits.

## Technologies Used
1.  NodeJS
2.  Express
3.  EJS
4.  MongoDB

## Prerequisites
- MongoDB
- Git
- NodeJS
- CLI

#### The Server runs at http://localhost:3000/
